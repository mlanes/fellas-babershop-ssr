# Typography

Two self-hosted families via `@fontsource/*` — no Google Fonts CDN.

- **Raleway** (400 / 600 / 700 / 800) — body and most headings
- **Bebas Neue** (400) — display accents (the hero title, the
  `FSectionHeading` headings get `text-h2` which uses Bebas)

> Source of truth: `assets/styles/_typography.scss`,
> `assets/styles/_variables.scss` ($font-primary).

## Scale

The `text-*` utility classes are global. Apply them to any element to
get the right size/weight/line-height — the _semantic_ tag (`<h1>`,
`<h2>`, `<p>`…) is chosen for meaning, the class for visual.

## Alignment helpers

`.text-center`, `.text-left`, `.text-right` — use sparingly; prefer
flex/grid alignment on parents.
